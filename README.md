# tuto-no-sql
Exo test sur MongoDB/Mongoose/NodeJS

## Prérequis
- NodeJS
- npm
- MongoDB
- Postman
- Robo3T (facultatif)

## Procédure

- cloner le projet
- lancer Mongo
- Créer une base MongoDB avec le nom du projet: tuto-no-sql
- npm install
- npm start
- Avec Postman, lancer la requete: 'GET http://localhost:4200/api/users'
